# 📚 Desafios Diários de JavaScript, React e Node.js

Repositório criado para documentar minha jornada de aprendizado no desenvolvimento web, realizando desafios práticos diários do zero com foco em entrar no mercado de trabalho como desenvolvedor Full Stack.

## 🚀 Sobre o projeto

Cada desafio foi desenvolvido do zero, praticando conceitos reais usados no mercado. Os desafios evoluem progressivamente — dos fundamentos do JavaScript até projetos completos com APIs, manipulação do DOM e lógica aplicada.

## 🛠️ Tecnologias utilizadas

- **JavaScript** — fundamentos, arrays, objetos, funções, async/await
- **HTML & CSS** — estrutura e estilização de interfaces
- **Node.js** — em andamento
- **React** — em breve
- **Git & GitHub** — versionamento de código

## 📂 Estrutura dos desafios

| Faixa | Conteúdo |
|-------|----------|
| 001 – 022 | Fundamentos JS: arrays, objetos, funções, filter, map, reduce |
| 023 – 044 | Manipulação do DOM com HTML e CSS |
| 045 – 054 | JSON, APIs e métodos avançados |
| 055 – 071 | Revisão geral e desafios completos com fetch e async/await |
